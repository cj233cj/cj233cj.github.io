Random game thoughts: "Ghost of Tsushima"

hugu 2026-04-23 Blog

‘ghost of tsushima’ has some serious issues with target locking when playing with M&K, but despite the pretty fucked up katana battle the game is very fun.

last time i enjoyed an action game with intense cold weapon combat was ‘assassin creed: origin’, ‘ghost of tsushima’ has a budget zelda-ish open world which is still much better and a very immersive lore/story if you don’t care about the history details;

the environment design is incredibly phenomenal, even in my potato PC;

music is very good also;

sucker punch really hate making visually attractive women, which is pretty racist and disrespectful tbf, that’s the only disappointment.

momiji higanbana ajisai, in the game weapons and armors are heavily connected with flora, resonating with ’the chrysanthemum and the sword’.
